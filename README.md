# cordova-plugin-wkwebviewxhrfix
Cordova plugin to fix local file access via XHR with WKWebView

## Installation

Install the plugin by running:
```
cordova plugin add https://github.com/TheMattRay/cordova-plugin-wkwebviewxhrfix
```
